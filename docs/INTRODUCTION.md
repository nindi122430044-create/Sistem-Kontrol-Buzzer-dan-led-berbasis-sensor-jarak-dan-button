# Sistem Kontrol Buzzer dan LED Berbasis Sensor Jarak dan Button

## Pendahuluan

Proyek "Sistem Kontrol Buzzer dan LED Berbasis Sensor Jarak dan Button" bertujuan membuat sistem interaktif yang memanfaatkan sensor jarak dan tombol untuk mengendalikan buzzer dan LED. Tujuan utamanya adalah mengembangkan mekanisme kontrol responsif yang mampu bereaksi terhadap kedekatan objek yang terdeteksi oleh sensor, serta memberikan umpan balik auditori dan visual kepada pengguna.

### Tujuan

Tujuan proyek ini adalah mendemonstrasikan integrasi komponen perangkat keras dengan pemrograman perangkat lunak untuk menghasilkan prototipe fungsional. Dengan menggunakan platform Arduino, kami ingin memberikan pengalaman praktis dalam desain sistem embedded, khususnya pemrosesan data sensor dan pengendalian output.

### Sasaran

1. Merancang dan mengimplementasikan sistem yang dapat mendeteksi jarak objek menggunakan sensor.
2. Mengendalikan buzzer dan LED berdasarkan pembacaan sensor dan input tombol.
3. Mengembangkan antarmuka pengguna yang sederhana untuk interaksi dengan sistem.
4. Mendokumentasikan proses pengembangan, kendala yang ditemui, dan solusi yang diterapkan.

### Teknologi yang Digunakan

- **Arduino**: Platform mikrokontroler utama untuk pemrograman dan pengendalian perangkat keras.
- **Sensor Ultrasonik**: Digunakan untuk mengukur jarak objek.
- **Buzzer**: Memberikan umpan balik suara sesuai kondisi sensor.
- **LED**: Memberikan umpan balik visual kepada pengguna.
- **Button (Tombol)**: Memungkinkan interaksi manual dengan sistem.

Proyek ini berfungsi sebagai sarana pembelajaran untuk memahami prinsip integrasi sensor, sistem kontrol, dan desain antarmuka pengguna pada sistem embedded.
